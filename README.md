# V1: сайт управления задачами руководителя

Простой веб-интерфейс для постановки задач голосом, делегирования и контроля статусов.

## Пользователи и invite-ссылки

- Руководитель (Имбицалов И.И.): `index.html?invite=inv-manager-001`
- Греховник А.А.: `index.html?invite=inv-grehovnik-001`
- Забияка В.В: `index.html?invite=inv-zabiyaka-001`
- Лошкатова С.С.: `index.html?invite=inv-loshkatova-001`
- Тупанкова Л.Л.: `index.html?invite=inv-tupankova-001`

Если invite не передан, открывается роль руководителя.

## Что реализовано в V1

- Экран руководителя "Быстрая постановка задачи":
  - Запись голосом (Web Speech API).
  - Автозаполнение текста задачи.
  - Автоподбор темы и исполнителя с ручной корректировкой.
  - Отправка задачи.
- Экран списка задач:
  - Фильтры: активные / выполненные / отмененные.
  - Разделение представления для руководителя и исполнителя.
- Экран деталей задачи:
  - Поля задачи и история событий.
  - Отмена задачи только руководителем.
  - Делегирование и изменение статуса исполнителем.

## Ограничения и правила доступа

- Задачи видит:
  - руководитель,
  - текущий исполнитель,
  - участники цепочки делегирования через историю.
- Исполнитель не может отменить задачу.
- Отмена доступна только руководителю.
- Вся логика V1 хранится в `localStorage` браузера.

## Критерии приемки V1

- Руководитель может поставить задачу за 2 действия:
  - записать голос,
  - отправить задачу.
- После голосового ввода система автоматически выставляет текст, тему и исполнителя.
- Исполнитель может делегировать задачу другому исполнителю.
- Отмена задачи работает только для руководителя.
